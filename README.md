Expense Tracker - ASP.NET Core MVC
📌 Project Overview
Expense Tracker is a web application built using ASP.NET Core MVC that helps users efficiently manage their income and expenses. It allows users to track transactions, categorize spending, and maintain financial control in a simple and structured way.

🚀 Features


➕ Add income and expense transactions


🗂️ Categorize transactions (Food, Travel, Bills, etc.)


✏️ Edit and delete transactions


📊 View all transactions in a structured list


💾 Data stored using SQL Server with Entity Framework Core


🔐 User-specific data handling (if implemented)


📱 Responsive UI using Bootstrap



🛠️ Tech Stack


Frontend: HTML, CSS, Bootstrap


Backend: ASP.NET Core MVC (.NET)


Database: Microsoft SQL Server


ORM: Entity Framework Core


Architecture: MVC (Model-View-Controller)
